GType panel_a11y_query_accessible_parent_type ( GType type,
GTypeInfo * type_info )

panel_a11y_query_accessible_parent_type @type: widget type @type_info: accessible object type info to complete

Standard hack which figures out the #GType of the accessible object for @type's parent type. Also, fills out the class_size and instance_size of @type_info to match the size of the parent type.

Basically, this is the hack to use if you want to derive from an accessible object implementation in gail.

Returns: the #GType of @type's parent's accessible peer
