msd_timeline_set_progress_func: @timeline: A MsdTimeline @progress_func: progress function

Sets the progress function. This function will be used to calculate a different progress to pass to the ::frame signal based on the linear progress through the timeline. Setting progress_func to NULL will make the timeline use the default function, which is just a linear progress.

All progresses are in the [0.0, 1.0] range.
