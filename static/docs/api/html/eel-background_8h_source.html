31/* Windows for Eel can contain backgrounds that are either tiled
32 with an image, a solid color, or a color gradient. This class manages
33 the process of loading the image if necessary and parsing the string
34 that specifies either a color or color gradient.
35
36 The color or gradient is always present, even if there's a tiled image
37 on top of it. This is used when the tiled image can't be loaded for
38 some reason (or just has not been loaded yet).
39
40 The EelBackground object is easier to modify than a GtkStyle.
41 You can just call eel_get_window_background and modify the
42 returned background directly, unlike a style, which must be copied,
43 modified and then set.
44*/
45
