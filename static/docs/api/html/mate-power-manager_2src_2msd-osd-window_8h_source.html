31/* MsdOsdWindow is an "on-screen-display" window (OSD). It is the cute,
32 * semi-transparent, curved popup that appears when you press a hotkey global to
33 * the desktop, such as to change the volume, switch your monitor's parameters,
34 * etc.
35 *
36 * You can create a MsdOsdWindow and use it as a normal GtkWindow. It will
37 * automatically center itself, figure out if it needs to be composited, etc.
38 * Just pack your widgets in it, sit back, and enjoy the ride.
39 */
40
