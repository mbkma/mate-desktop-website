33/*
34 * SECTION: cact-tree-view
35 * @title: CactTreeView
36 * @short_description: The Tree View Base Class Definition
37 * @include: cact-tree-view.h
38 *
39 * This is a convenience class to manage a read-only items tree view.
40 *
41 * The CactTreeView encapsulates the GtkTreeView which displays the items
42 * list on the left of the main pane.
43 *
44 * It is instanciated from CactMainWindow::on_initialize_gtk().
45 *
46 * A pointer to this CactTreeView is attached to the CactMainWindow at
47 * construction time.
48 */
