mateweather_parser_get_localized_value: @parser: a MateWeatherParser

Looks at the name of the element @parser is currently pointing to, and returns the content of either that node, or a following node with the same name but an "xml:lang" attribute naming one of the locale languages. Leaves @parser pointing to the next node after the last consecutive element with the same name as the original element.

Return value: the localized (or unlocalized) text, as a libxml-allocated string, or NULL if the node is empty.
