void(* BaseWindowClass::initialize_base_window) (BaseWindow *window)

initialize_base_window: @window: this BaseWindow instance.

This virtual method is invoked as the first phase of base_window_init(), after having checked for the presence of a GtkWindow toplevel, before actually displaying the widget, and after all connected signal handlers have themselves run.

The derived class should invoke the virtual method of its parent class at the end of its processing.

The BaseWindow base class implementation of this method, which is so called last, just does nothing.

◆ initialize_gtk_toplevel

void(* BaseWindowClass::initialize_gtk_toplevel) (BaseWindow *window, GtkWindow *toplevel)

initialize_gtk_toplevel: @window: this BaseWindow instance. @toplevel: the GtkWindow being initialized.

This virtual method is invoked after the toplevel GtkWindow has been allocated and built for the firt time by the GtkBuilder, after all connected signal handlers have themselves run.

The derived class should invoke the virtual method of its parent class at the end of its processing.

The BaseWindow class ensures that each created instance has its Gtk toplevel - if it has been successfully loaded - initialized when returning from instanciation. The instance has nonetheless still to check if the Gtk toplevel has actually been built, and to initialize and show it before anything useful may occur (see base_window_init()).

Note that initialization process may fall if the XML UI definition cannot be loaded in memory, or if the required Gtk toplevel cannot be found. Derived class has so to make sure that a Gtk toplevel actually exists before continuing. Calling base_window_init() on the instance may do this check.

int(* BaseWindowClass::run) (BaseWindow *window)

run: @window: this BaseWindow instance. @dialog: the toplevel #GtkWindow.

Invoked when it is time to run the main loop for the toplevel if the derived class does not rely on the global main loop.

The BaseWindow -derived instance should implement this method if it wants do something special.

Returns: the exit code as set by the derived class.

void(* BaseWindowClass::show_widgets) (BaseWindow *window)

show_widgets: @window: this BaseWindow instance.

This virtual method is invoked at the end of initialization process, after all connected signal handlers have themselves run.

The derived class should invoke the virtual method of its parent class at the end of its processing.

The BaseWindow base class implementation of this method, which is so called last, will call gtk_widget_show_all() on the Gtk toplevel.
