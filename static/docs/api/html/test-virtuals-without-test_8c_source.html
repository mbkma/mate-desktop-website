30/* We want test here what is the exact behavior of virtual functions in
31 * derived classes, whether or not base class has implemented them or
32 * not.
33 *
34 * We define three classes, and some virtual functions :
35 * class A: fn1, fn2, fn3
36 * class AB: implements fn1, fn2
37 * class ABC: implements fn1, fn3
38 *
39 * Public entry points are defined in class A: we check that calling
40 * public entry points with an object of each class actually calls the
41 * relevant virtual function.
42 *
43 * Also we check that calling the parent class is possible even if the
44 * parent class has not explicitely defined the virtual function.
45 *
46 * Same as test-virtuals.c, without the test for existance of function.
47 */
