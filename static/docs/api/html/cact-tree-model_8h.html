#define CACT_TYPE_TREE_MODEL   ( cact_tree_model_get_type())

SECTION: cact_tree_model @short_description: CactTreeModel class definition. cact/cact-tree-model.h

CactTreeModel is derived from GtkTreeModelFilter in order to be able to selectively display profiles, whether an action has one or more profiles.

CactTreeModel implements EggTreeMultiDragSource and GtkTreeDragDest interfaces.

The GtkTreeModelFilter base class embeds a GtkTreeStore.
