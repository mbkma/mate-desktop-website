◆ eel_labeled_image_new()

GtkWidget * eel_labeled_image_new ( const char * text,
GdkPixbuf * pixbuf )

eel_labeled_image_new: @text: Text to use for label or NULL. @pixbuf: Pixbuf to use for image or NULL.

Returns A newly allocated EelLabeledImage. If the &text parameter is not NULL then the LabeledImage will show a label. If the &pixbuf parameter is not NULL then the LabeledImage will show a pixbuf. Either of these can be NULL at creation time.

Later in the lifetime of the widget you can invoke methods that affect the label and/or the image. If at creation time these were NULL, then they will be created as neeeded.

Thus, using this widget in place of EelImage or EelLabel is "free" with only the GtkWidget and function call overhead.
