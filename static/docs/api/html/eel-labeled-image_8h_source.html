25/* EelLabeledImage is a container widget. It can only contain internal
26 * widgets. These internal widgets are can be a EelLabel and/or a
27 * EelImage. These internal widgets are created as needed. That means
28 * that EelLabeledImage can always be used for "free" instead of a
29 * EelLabel or EelImage. The only overhead is that of the GtkObject
30 * machinery.
31 *
32 * The position of the label with respect to the image is controlled by the
33 * 'label_positon' attribute.
34 *
35 * By default the internal image and label widgets are sized to their natural
36 * preferred geometry. You can use the 'fill' attribute of LabeledImage
37 * to have the internal widgets fill as much of the LabeledImage allocation
38 * as is available.
39 *
40 * LabeledImage also has x_padding/y_padding and x_alignment/y_alignment
41 * attributes that behave exaclty as those in the GtkMisc class.
42 *
43 * Note that the alignments are ignored if the fill attribute is TRUE.
44 */
45
