mate_rr_screen_refresh: @screen: a MateRRScreen @error: location to store error, or NULL

Refreshes the screen configuration, and calls the screen's callback if it exists and if the screen's configuration changed.

Return value: TRUE if the screen's configuration changed; otherwise, the function returns FALSE and a NULL error if the configuration didn't change, or FALSE and a non-NULL error if there was an error while refreshing the configuration.
