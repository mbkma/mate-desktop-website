Detailed Description

NAIFactoryProviderInterface: @get_version: returns the version of this interface the plugin implements. @read_start: triggered just before reading an item. @read_data: reads an item. @read_done: triggered at the end of item reading. @write_start: triggered just before writing an item. @write_data: writes an item. @write_done: triggered at the end of item writing.

This defines the interface that a NAIFactoryProvider may implement.
