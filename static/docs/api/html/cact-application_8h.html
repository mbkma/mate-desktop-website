SECTION: cact_application @short_description: CactApplication class definition. cact/cact-application.h

This is the main class for caja-actions-config-tool program.

The CactApplication object is instanciated in main() function.

Properties are explicitely set in cact_application_new() before calling base_application_run().

The CactApplication object is later g_object_unref() in main() after base_application_run() has returned.
