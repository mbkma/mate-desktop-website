7 <name>Engrampa Archive Manager</name>
8 <summary>An Archive Manager for the MATE desktop environment</summary>
9 <description>
10 <p>
11 Engrampa is an archive manager for the MATE environment. It allows you
12 to create and modify archives, view the contents of an archive, view a
13 file contained in an archive, and extract files from archive.
14 </p>
15 <p>
16 Engrampa is only a front-end (a graphical interface) to archiving programs
17 like tar and zip. The supported file types are:
18 </p>
19 <ul>
20 <li>7-Zip Compressed File (.7z)</li>
21 <li>WinAce Compressed File (.ace)</li>
22 <li>ALZip Compressed File (.alz)</li>
23 <li>AIX Small Indexed Archive (.ar)</li>
24 <li>ARJ Compressed Archive (.arj)</li>
25 <li>Cabinet File (.cab)</li>
26 <li>UNIX CPIO Archive (.cpio)</li>
27 <li>Debian Package (.deb, .udeb) [Read-only mode]</li>
28 <li>ISO-9660 Disk Image (.iso) [Read-only mode]</li>
29 <li>Java Archive (.jar)</li>
30 <li>Java Enterprise archive (.ear)</li>
31 <li>Java Web Archive (.war)</li>
32 <li>LHA Archive (.lzh, .lha)</li>
33 <li>WinRAR Compressed Archive (.rar)</li>
34 <li>RAR Archived Comic Book (.cbr)</li>
35 <li>RPM Package Manager (.rpm) [Read-only mode]</li>
36 <li>Tape Archive File uncompressed (.tar) or compressed with: gzip (.tar.gz, .tgz), brotli (.tar.br), bzip (.tar.bz, .tbz), bzip2 (.tar.bz2, .tbz2),
37 compress (.tar.Z, .taz), lrzip (.tar.lrz, .tlrz), lzip (.tar.lz, .tlz), lzop (.tar.lzo, .tzo), 7zip (.tar.7z), xz (.tar.xz), Zstandard (.zst)
38 </li>
39 <li>Stuffit Archives (.bin, .sit)</li>
40 <li>ZIP Archive (.zip)</li>
41 <li>ZIP Archived Comic Book (.cbz)</li>
42 <li>ZOO Compressed Archive File (.zoo)</li>
43 <li>Single files compressed with gzip (.gz), brotli (.br), bzip (.bz), bzip2 (.bz2), compress (.Z),
44 lrzip (.lrz), lzip (.lz), lzop (.lzo), rzip(.rz), xz (.xz).</li>
45 </ul>
46 <p>
47 Engrampa is a fork of File Roller and part of the MATE Desktop Environment.
48 If you would like to know more about MATE and Engrampa, please visit the
49 project's home page.
50 </p>
51 </description>
