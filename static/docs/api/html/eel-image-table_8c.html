signal_connect_while_realized:

@object: Object to connect to.

@callback: Caller's callback. @callback_data: Caller's callback_data. @realized_widget: Widget to monitor for realized state. Signal is connected while this wigget is realized.

Connect to a signal of an object while another widget is realized. This is useful for non windowed widgets that need to monitor events in their ancestored windowed widget. The signal is automatically disconnected when &widget is unrealized. Also, the signal is automatically disconnected when either &object or &widget are destroyed.
