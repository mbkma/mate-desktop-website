void mate_bg_crossfade_start ( MateBGCrossfade * fade,
GdkWindow * window )

mate_bg_crossfade_start: @fade: a MateBGCrossfade @window: The #GdkWindow to draw crossfade on

This function initiates a quick crossfade between two surfaces on the background of @window. Before initiating the crossfade both mate_bg_crossfade_set_start_surface() and mate_bg_crossfade_set_end_surface() need to be called. If animations are disabled, the crossfade is skipped, and the window background is set immediately to the end surface.
