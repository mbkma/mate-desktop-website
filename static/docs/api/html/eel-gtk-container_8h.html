eel_gtk_container_child_expose_event:

@container: A GtkContainer widget. @child: A child of @container or NULL; @event: The expose event.

Forward an expose event to a child if needed. It is valid to give a NULL @child. In that case this function is a noop. Proper clipping is done to ensure that the @child does indeed need to be forwarded the exposure event. Finally, the forwarding only occurs if the child is a NO_WINDOW widget. Of course, it is valid to feed non NO_WINDOW widgets to this function, in which case this function is a noop.
